# Desafio 01
  Esse desafio vem com uma proposta de ser um blog bem simples, com algumas postagens contendo
  titulo, data de publicação, descrição e um coração para dar um favoritar ou like.
  
## Experiência
  Com este projeto pratiquei mais o conceito de responsividade, criatividade, pois eu tive a ideia de colocar
  a funcionalidade de favoritar/gostar da postagem. Achei incrível a maneira que desenvolvi e que organizei meu código.

## Tecnologias utilizadas
  * HTML
  * CSS
  * JS
